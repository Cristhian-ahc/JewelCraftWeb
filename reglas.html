Reglas y términos de uso

Reglas generales Reglas del servidor Reglas del discord Términos de uso

Reglas generales

  • 1. Respetar al resto de usuarios, no se permite el acoso o abuso de ningún tipo, incluído filtrar información personal, extorsión u hostigar
  • 2. No se permite el contenido nsfw o ilegal, dentro de esto se encuentra contenido sexual explícito, gore, sangriento o que cause molestia a otros usuarios
  • 3. No se permite ningún tipo de spam, dentro de esto se encuentra flood, publicidad, abuso de mayúsculas
  • 4. No se permite usurpar identidades con fines perjudiciales
  • 5. Respeta el uso de canales respectivamente
  • 6. Cualquier duda, consulta, reclamo, sugerencia, reportes o denuncias debe hacerce principalmente mediante tickets en soporte o bien con cualquier staff disponible, se tratara de solucionar los problemas a la brevedad
  • 7. Está prohibido todo tipo de estafa o scam
  • 8. Las multicuentas estan prohibidas en caso sea para sacar ventaja sobre otros jugadores,con fines maliciosos o para saltarte sanciones
  • 9. Se ético y usa el sentido común 

Reglas del servidor

  • Las reglas dichas anteriormente y las siguientes reglas son aplicables para todas las modalidades exceptuando anarchy
  • 1. Se prohíben todo tipo de hack, mods, clientes modificados y paquetes de texturas con fines malicioso o para tener ventaja sobre el resto de los jugadores, dentro de estos se incluyen modificaciones que arruinen la experiencia de juego de la comunidad tales como vuelo, reach, velocity o knockback reducer, obtención y clonación de objetos y entidades
  • 1.1 se permiten clientes, modificaciones o textures packs que sirvan para optimización del juego o comodidad de interfaz, tales cómo OptiFine o clientes como lunar o badlion
  • 1.2 las modificaciones de minimapa están permitidas dependiendo de la modalidad
  • 2. Está prohibido abusar o sacar provecho de errores del juego o del servidor, cualquier detección de errores debe ser reportado al staff
  • 2.1 se hace exclusión algunos Bugs que ya forman parte del estilo de juego, tales cómo subir al techo del nether o sacar bedrock
  • 3 está prohibido pedir dinero o mendigar objetos a otros usuarios
  • 4 tanto skin, nombre de cuenta, nombres de objetos u otros deben ser apropiados
  • 5 está prohibido el tpakill, el spawnkill y el spamkill
  • 6. Las granjas de gran tamaño están permitas siempre y cuando no genere gran latencia
  • 6.1 otras construcciones que generen latencia estan bajo las mismas reglas (item sorters)
  • 6.2 las granjas 0 ticks están prohibidas
  • 6.3 si esto ocurre las acciones varían dependiendo del caso, desde devolución de objetos hasta desactivar la granja momentáneamente

Reglas del discord

  • 1. No se permite el acoso o abuso de ningún tipo.
  • 2. No se permite contenido con restricción por edad ni obsceno.
  • 3. No se permite el spam ni la autopromoción.
  • 4. No se permite usurpar identidades con fines perjudiciales.
  • 5. El desconocimiento de las reglas no justifica el incumplimiento de las mismas.
  • 6. Respeta el uso de los canales respectivamente.
